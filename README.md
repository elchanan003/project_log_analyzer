# מנתח לוגים של תעבורת רשת

מערכת לניתוח קובצי לוג של תעבורת רשת, המזהה פעילות חשודה ומייצרת דוח מסכם.

המערכת קוראת קובץ לוג, מזהה מספר סוגי חשדות שונים בפעילות הרשת, אוספת סטטיסטיקות ומייצרת דוח מסודר של כתובות IP חשודות.

הפרויקט בנוי בצורה מודולרית ומדגים עבודה עם:
- Generators
- עיבוד נתונים
- חלוקה למודולים
- הפקת דוחות

---

# יכולות המערכת

המערכת מסוגלת:

- לקרוא קובץ לוג בכל גודל בעזרת Generators
- לזהות פעילות חשודה לפי מספר קריטריונים
- לאסוף סטטיסטיקות בזמן אמת
- ליצור דוח מסכם של פעילות חשודה
- לשמור את הדוח לקובץ טקסט

---

# סוגי החשדות שהמערכת מזהה

המערכת מזהה ארבעה סוגי פעילות חשודה:

**EXTERNAL_IP**  
גישה מכתובת IP חיצונית שאינה מהרשת הפנימית.

**SENSITIVE_PORT**  
גישה לפורטים רגישים כגון:
- 22 (SSH)
- 23 (Telnet)
- 3389 (Remote Desktop)

**LARGE_PACKET**  
חבילות מידע גדולות במיוחד.

**NIGHT_ACTIVITY**  
פעילות שמתרחשת בשעות הלילה.

# מבנה הפרויקט

```

project_log_analyzer/

reader.py
checks.py
analyzer.py
reporter.py
config.py
main.py

network_traffic.log

```

תפקיד הקבצים:

**reader.py**  
קריאת קובץ הלוג.

**checks.py**  
בדיקות לזיהוי סוגי החשדות.

**analyzer.py**  
ניתוח הלוג, עדכון סטטיסטיקות ובניית מילון חשודים.

**reporter.py**  
יצירת דוח ושמירתו לקובץ.

**config.py**  
הגדרות וקבועים של המערכת.

**main.py**  
הפעלת כל המערכת.
